# Skill Pack: Visual Strategy Rubric (v1)

Read visuals as strategic choices, not aesthetics. Every observation must answer: *what marketing work does this do?*

## Production quality as a signal (not a grade)

- **Raw / lo-fi** (phone camera, screenshot aesthetic, no grading) → authenticity play; "a person like you posted this", dodges ad-blindness.
- **Mid / template** (Canva-grade, brand fonts, stock icons) → volume content; consistency over impact.
- **Polished / cinematic** → premium positioning, trust-building for high-ticket; also higher ad-skepticism.
- Mismatch is information: polished brand posting deliberately raw = borrowing creator credibility.

## Layout & hierarchy

- First fixation point (face > big number > high-contrast text > product). Where does the eye land, and is that where the message starts?
- Text density: word count of the first frame; mobile legibility at thumbnail size.
- Faces: direction of gaze steers attention; emotion on the face pre-loads the emotion of the copy.
- Thumbnail energy: would this frame earn a tap at 150×260 px? Why?

## Palette (one signal among many — never the headline)

Note palette only when it does marketing work:
- **Category convention** (edtech blues/purples, finance greens, alert reds) — conforming or breaking, and why.
- **Attention contrast** — complementary punch against the platform's UI chrome (Instagram dark mode, LinkedIn white).
- **Brand recall** — consistent recognizable color = feed signature (Blinkit yellow, Zomato red).
- **Emotional temperature** — warm urgency vs. cool authority.
The programmatic palette (hex + % coverage) is provided to you; interpret it, don't recite it.

## Format conventions

- 9:16 / 1:1 / 4:5 / 16:9 — what placement is this built for, and does the composition respect that placement's safe zones (captions, UI buttons)?
- Carousel: does slide 1 work alone? Is there a swipe-forcing device (cut-off sentence, arrow, cliffhanger)?
- Motion (if inferable from stills): jump-cut pacing, captions burned in = sound-off design.

## Anti-checklist (do not do)

- No "aesthetically pleasing", "eye-catching", "clean design" without the mechanism behind it.
- No palette poetry ("calming lavender evokes serenity") unless tied to a marketing function.
- No inventing what can't be seen (motion, sound) — mark inferences as inferences.
