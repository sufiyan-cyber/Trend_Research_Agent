# Skill Pack: Campaign Play Taxonomy (v1)

Every campaign runs one primary play, sometimes with a secondary. Name it precisely.

| Play | Signature | Tell-tale evidence |
|---|---|---|
| **Launch / Drop** | New thing + moment engineering | Countdown, "introducing", waitlist, teaser sequence |
| **FOMO / Scarcity** | Limited time/seats/stock | Deadlines, "only N left", expiring discount, cohort close date |
| **Social proof** | Others already chose this | Testimonials, user counts, logos, ratings, "join 10,000+" |
| **UGC bait** | Designed to be remade/shared | Template formats, challenges, duet/stitch hooks, "show us your…" |
| **Newsjack / Moment marketing** | Rides a live event or news cycle | References to matches, elections, weather, trending memes — timing IS the idea |
| **Seasonal** | Calendar-anchored | Festivals, exam season, placement season, New Year, budget day |
| **Founder story / Build in public** | Person over brand | First-person narration, vulnerability, numbers shared openly |
| **Controversy / Contrarian** | Picks a fight with consensus | "Unpopular opinion", attacks a common practice, comment-bait framing |
| **Meme hijack** | Borrows a circulating format | Recognizable template, low production, speed over polish |
| **Community flex** | Belonging as product | Inside jokes, member spotlights, "if you know you know" |
| **Nostalgia** | Emotional time travel | Retro visuals, "remember when", childhood references |
| **Us-vs-them** | Identity through opposition | "Unlike X…", category villain, switching narratives |
| **Education-first / Value bomb** | Teach to earn trust | Carousels of tips, free mini-course, "save this post" |
| **Status / Aspiration** | Buyer becomes the upgraded self | Before/after, salary screenshots, lifestyle imagery |

## Channel fingerprints (infer channel from format)

- **9:16 vertical, big captions, fast cuts** → Reels/Shorts/TikTok
- **Text-dense carousel, professional tone** → LinkedIn
- **Square, single-frame, minimal text** → Instagram feed
- **Long copy + single link, conversational** → email/newsletter
- **Screenshot aesthetic (chat, notes app, tweet)** → cross-platform virality play, authenticity signal
- **Poster with QR code** → offline/event bridge

## Timing logic checklist

Ask: why did this run NOW? Product moment / cultural moment / seasonal calendar / competitor response / algorithm trend-window. If no timing logic is visible, say so — "evergreen" is also an answer.

## CTA structures

Hard CTA (buy/register + urgency) · Soft CTA (follow/save/share) · Engagement bait (comment a word, tag someone) · Lead magnet (DM for the file) · Deliberate absence (pure brand impression). Note which one and why it fits the funnel stage.
