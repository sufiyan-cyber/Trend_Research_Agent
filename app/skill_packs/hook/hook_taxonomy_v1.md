# Skill Pack: Hook & Copy Taxonomy (v1)

The hook is whatever the audience meets first: headline, first line, first 1–2 seconds, thumbnail text. Identify the mechanic by name and quote the actual text.

## Hook mechanics

| Mechanic | How it stops the scroll | Example shape |
|---|---|---|
| **Curiosity gap** | Opens a loop the brain must close | "Nobody talks about the real reason…" |
| **Bold claim** | Big promise, dares you to doubt it | "This got me 3 offers in 30 days" |
| **Specificity** | Precise numbers feel true | "₹37,450" beats "good money"; "day 47" beats "recently" |
| **Self-selection callout** | Names the reader, filters the feed | "If you're a final-year CS student…" |
| **Pattern interrupt** | Breaks the feed's visual/verbal rhythm | Unexpected format, taboo word, weird image |
| **Stakes / Loss frame** | What you lose by scrolling past | "Every day you wait, the gap grows" |
| **Humor / Absurdity** | Laughter lowers guard, earns the share | Exaggeration, deadpan, meme logic |
| **Authority borrow** | Credibility transferred from a bigger name | "Google engineer explains…", press logos |
| **Confession / Vulnerability** | Honesty is rare in feeds, so it reads as signal | "I failed 6 interviews before I understood this" |
| **Question trap** | A question the reader must answer internally | "Would you take ₹12 LPA at 22?" |
| **List promise** | Bounded effort, guaranteed structure | "5 prompts that…" |

## Credibility devices (note when present)

Numbers with sources · screenshots as receipts · named people with faces · timestamps · third-party logos · community size · "no-edit" raw footage claims.

## Copy structure checks

- **Line 1 → Line 2 handoff:** does the first line force the second? (Good hooks are cliffhangers, not summaries.)
- **Reading gravity:** short lines, line breaks as pacing, one idea per line = built for mobile skim.
- **Voice:** first-person confession / second-person address ("you") / observer narration — which, and does it match the play?
- **Jargon budget:** who does the vocabulary include or exclude on purpose?

## Emotion map (pick the dominant one)

FOMO · aspiration · belonging · insecurity ("everyone's ahead of you") · pride · outrage · relief ("finally someone said it") · curiosity · amusement · nostalgia. Name the exact word/image that fires it.
